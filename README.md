

# Django Todo ORM

## 🇷 Русская версия

Простое CRUD‑приложение на Django ORM.

### Функции
- создание задач
- просмотр задач
- изменение статуса выполнения
- удаление задач

### Модель

Task:
- title — название задачи
- description — описание
- completed — выполнена ли задача
- created_at — дата создания

### Установка

```bash
git clone https://github.com/rakhim-musanipov/django-todo-orm.git
cd django-todo-orm
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
